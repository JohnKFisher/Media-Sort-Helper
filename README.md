# Media Sort Helper (macOS)

This repo is retired.

`Media-Sort-Helper` was an early folder-review experiment. Its replacement is [Photos Library Sort Helper](https://github.com/JohnKFisher/Photos-Library-Sort-Helper), which now covers both:

- Apple Photos library review
- regular folder-of-files review

No new feature work should land here. If you want the current app, use the replacement repo instead:

- https://github.com/JohnKFisher/Photos-Library-Sort-Helper

## Why This Repo Was Retired

The newer app has the better shell, safer workflow, shared review UI, session persistence, better packaging/release structure, and now supports folder mode too. Keeping both apps alive would just split future work across two very similar codebases.

## Status

- Deprecated and retired
- Kept for history/reference only
- Not the recommended download or contribution target

## Replacement

Use `Photos Library Sort Helper`:

- Repo: https://github.com/JohnKFisher/Photos-Library-Sort-Helper
- Current direction: one macOS app with equal `Photos Library` and `Folder` source modes

## Historical Note

This repo originally focused on top-level folder scanning and moving reviewed files into opinionated sibling folders. That behavior is no longer the preferred path forward.
